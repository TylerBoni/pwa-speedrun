# 1. Introduction

## Free Accounts

- **GitHub** - Version control and repository hosting
- **Cloudflare** - Domain registration
- **Supabase** - Backend as a Service
  - Max 2 active projects
  - No branches in free plan
- **Lovable** - AI Website builder
  - 5 credits daily
  - 1 new project per day
  - All projects are PUBLIC
- **Google Cloud** - Authentication
- **Vercel** - Application hosting

## Paid Accounts

- **Cursor** - IDE Code editor ($20/mo)

## Software

- **IDE**: Cursor
- **Function Deployment**: Supabase CLI
- **JavaScript Runtime**: Node
- **Package Manager**: PNPM
- **Optional**:
  - **GH CLI** - GitHub command-line tool for creating repos

## Tech Stack

- **Framework**: Vite + React
- **Supabase**:
  - Auth
  - Database: Postgres
  - API: PostgREST

## Concepts to Understand

- **RLS (Row Level Security)**: Rules that manage access to database tables
- **OAuth**: Using a secondary authentication provider
